# Mood Indigo 2005

Mood Indigo 2005 back online after 21 years.

Live: <https://saurabhdas.github.io/mi05/>

Each versioned build is also addressable at its own path:

- <https://saurabhdas.github.io/mi05/ruffled1/> — HTML-overlay nav, two Ruffle players (currently default)
- <https://saurabhdas.github.io/mi05/ruffled2/> — single-player rebuild (button bytecode patched, experimental)

The default version served at the site root is configurable via `DEFAULT_VERSION` in [`.github/workflows/pages.yml`](.github/workflows/pages.yml).
